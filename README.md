# TypeScript
### O que é?
- É um superset do JavaScript
- Types, features de versões futuras do JavaScript, Interfaces, Generics e Decorations
- Utiliza-se em conjunto de outras tecnologias como o React 

### Instalação
`npm install -g typescipt`

### .ts para .js
`npx tsc app.ts`
